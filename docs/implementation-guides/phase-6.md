# Phase 6: Informational Pages & SEO Guide

## 1. About Us
Create `src/app/about/page.tsx`:
- Detailed narrative of "Silver Spoon".
- Showcase craftsmanship and the heritage of silver gifting.
- Use elegant GSAP-driven scroll animations for text blocks.

## 2. Contact Us
Create `src/app/contact/page.tsx`:
- Simple contact form (Name, Email, Phone, Message).
- Direct WhatsApp link for quick queries.
- Store location details if applicable.

## 3. SEO Strategy
- **Metadata API:** Use Next.js `generateMetadata` for dynamic product and category pages.
- **Keywords:** Silver jewellery, Pure silver gifts, 999 Silver Frames, Sterling silver idols, Luxury silver gifting India.
- **Open Graph:** Set up high-quality images for social sharing.
- **Sitemap:** Generate `sitemap.xml` using `next-sitemap` or custom script.
